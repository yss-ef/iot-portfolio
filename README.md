# IoT portfolio

This repository contains a specialized collection of Internet of Things (IoT)
projects and laboratories. The work focuses on secure communication protocols,
industrial connectivity (IIoT), and digital twin modeling.

## Executive summary: industry 4.0 and security

This portfolio demonstrates expertise in the MQTT ecosystem, ranging from
foundational sensor simulations to hardened industrial frameworks. Key
implementations include Mutual TLS (mTLS) for encrypted telemetry and
physics-based digital twins that simulate thermal dynamics for predictive
maintenance.

## Repository structure

### Labs

- [Secure MQTT TLS](./labs/secure-mqtt-tls): Hardening MQTT communication using
  X.509 certificate chains, implementing mutual authentication and Last Will
  and Testament (LWT) patterns.
- [Sensor Simulation](./labs/sensor-simulation): Foundational pub/sub
  simulations for telemetry streaming and broker performance testing.

### Projects

- [Industrial Digital Twin](./projects/industrial-digital-twin): An Industry 4.0
  simulation featuring multi-threaded telemetry engines, thermal inertia
  modeling, and real-time dashboard monitoring.

## Tech stack

- Protocols: MQTT (Paho-MQTT), SSL/TLS (mTLS)
- Languages: Python (Multi-threading, Numerical Modeling)
- Broker: Eclipse Mosquitto
- Security: X.509 certificates, asymmetric encryption
