# IoT Portfolio

A specialized collection of Internet of Things (IoT) projects and laboratories, focusing on secure communication protocols, industrial connectivity (IIoT), and digital twin modeling.

## Executive Summary: Industrial 4.0 & Security
This portfolio demonstrates advanced expertise in the MQTT ecosystem, transitioning from basic sensor simulations to hardened industrial frameworks. Key implementations include **Mutual TLS (mTLS)** for encrypted telemetry and **Physics-based Digital Twins** that simulate real-world thermal dynamics for predictive maintenance.

## Repository Structure

### Labs
*   **[Secure MQTT TLS](./labs/secure-mqtt-tls)**: Hardening MQTT communication using X.509 certificate chains, implementing Mutual Authentication and Last Will and Testament (LWT) patterns.
*   **[Sensor Simulation](./labs/sensor-simulation)**: Foundational Pub/Sub simulations for telemetry streaming and broker performance testing.

### Projects
*   **[Industrial Digital Twin](./projects/industrial-digital-twin)**: An Industry 4.0 simulation featuring multi-threaded telemetry engines, thermal inertia modeling, and real-time dashboard monitoring.

## Tech Stack
*   **Protocols**: MQTT (Paho-MQTT), SSL/TLS (mTLS)
*   **Languages**: Python (Multi-threading, Numerical Modeling)
*   **Broker**: Eclipse Mosquitto
*   **Security**: X.509 certificatess, Asymmetric Encryption
